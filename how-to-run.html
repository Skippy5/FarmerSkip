How to Run Farmer Skip

Farmer Skip's Eggcellent Adventure uses ES6 modules, which requires the game to be served from an HTTP server rather than opened directly from the file system. Here are several ways to run the game:

Method 1: Using Python's Built-in Server (Recommended)

If you have Python installed (which most computers do):

For Python 3

python -m http.server

For Python 2

python -m SimpleHTTPServer

Then open your browser and navigate to: http://localhost:8000

Method 2: Using Visual Studio Code Live Server

If you use VS Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"

This will automatically open the game in your default browser.

Method 3: Using Node.js (If You Have It Installed)

If you have Node.js installed:

npx http-server

Or, install the server.js dependencies first:

npm install
npm start

Then open your browser and navigate to: http://localhost:8080

Troubleshooting

  • Make sure you're accessing the game through an HTTP server, not by opening the file directly.
  • Check the browser console (F12) for any error messages.
  • If you see CORS errors, it confirms you need to use an HTTP server.
